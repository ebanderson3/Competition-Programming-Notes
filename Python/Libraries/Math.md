```python
import math
```
## Number Theory
```python
math.gcd(a, b)        # greatest common divisor
math.lcm(a, b)        # least common multiple (3.9+)
math.isqrt(n)         # integer sqrt (floor(sqrt(n)))
math.factorial(n)     # n!
```
## Combinatorics
```python
math.comb(n, k)       # n choose k
math.perm(n, k)       # n P k
```
## General Math
```python
math.ceil(x), math.floor(x)
math.log(x, base)     # logarithm
math.sqrt(x)
math.pi, math.e
pow(base, expo, mod)
```
